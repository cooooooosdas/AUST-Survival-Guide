# 安理大生存指南

安理大生存指南是一个面向安徽理工大学新生的资源导航与经验分享平台。从工具箱到学长来信，从 AI 助手到资源下载，一站式覆盖新生入学的方方面面。

## 核心功能

- **工具箱**：精选学习、生活常用网站链接
- **学长来信**：长文经验分享，覆盖学业、生活、职业规划
- **AI 助手**：内置智能对话，先站内搜索再 AI 回退
- **资源下载**：笔记、课件、软件包集中下载
- **社区互动**：留言、点赞、收藏、评论

## 技术栈

| 层级 | 技术 |
|------|------|
| 框架 | Next.js 16 (App Router) |
| 语言 | TypeScript |
| 样式 | Tailwind CSS v4 |
| 数据库 | Supabase |
| 部署 | Vercel |

## 项目亮点

1. **暗色模式**：全站配色自适应，防闪烁实现
2. **移动端适配**：响应式布局 + 汉堡菜单 + 触控优化
3. **搜索系统**：站内全文搜索，关键词评分排序
4. **MDX 内容**：来信和项目展示均使用 MDX 格式
