# AI 学习助手

AI 学习助手是一个基于大模型的智能学习辅助工具，旨在帮助学生更高效地学习高等数学、编程等科目。

## 功能特性

- **数学题解答**：拍照或输入题目，获取详细解题步骤
- **代码调试**：粘贴报错信息，AI 分析并提供修复建议
- **学习路径规划**：根据目标推荐学习资源和顺序
- **笔记总结**：自动整理课程笔记，生成复习提纲

## 技术栈

- 后端：FastAPI + WebSocket 流式响应
- 前端：React + Tailwind CSS
- AI 模型：Claude API / GPT-4
- 部署：Docker + Nginx

## 开发心得

这个项目让我深入理解了如何在实际场景中调用大模型 API，特别是：
- Prompt Engineering 对回答质量的影响
- 流式响应的用户体验优化
- Token 成本控制策略
