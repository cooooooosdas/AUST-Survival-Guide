# 课程管理系统

课程管理系统是软件工程专业的一门课程大作业，实现了高校学生课程管理的基础功能。

## 功能模块

- **学生管理**：学生信息的增删改查
- **课程管理**：课程信息维护、选课人数控制
- **成绩管理**：成绩录入、查询、统计分析
- **课表生成**：自动检测冲突、生成个人课表

## 技术实现

- 前端使用 Vue 3 + Element Plus 构建响应式界面
- 后端使用 Node.js + Express 提供 RESTful API
- MySQL 存储数据，Sequelize ORM 管理模型
- JWT 认证、角色权限控制
