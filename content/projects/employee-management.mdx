# 员工信息管理系统

软件工程专业大一上学期 C 语言课程设计，结合 EasyX 图形库实现的桌面端员工信息管理系统。

## 项目背景

这是软工大一上学期 C 语言实训的大作业。要求用 C 语言 + EasyX 图形库做一个带图形界面的信息管理系统，覆盖结构体、文件读写、链表/数组操作等大一无知识点。

参考视频教程：[结合 Easyx 的信息管理系统？(附大作业源码)[c 语言实训]](https://www.bilibili.com/video/BV1eRiBBdEre/) —— 视频里是 UP 主"那位酷影"的实现版本，本项目在其基础上完成了员工信息管理这一具体场景。

## 功能模块

- **员工信息录入**：工号、姓名、部门、职位、薪资等字段
- **信息维护**：增、删、改、查
- **查询统计**：按工号 / 姓名模糊查询，按薪资 / 部门统计
- **文件持久化**：所有数据落盘到本地文件，重启程序后仍可读取
- **图形界面**：EasyX 绘制菜单、表单、表格，鼠标 + 键盘交互

## 技术栈

- **C 语言** —— 核心逻辑
- **EasyX 图形库** —— Windows 下的轻量 2D 绘图库，配合 Visual Studio 使用
- **文件 I/O** —— 二进制 / 文本文件持久化

## 收获

- 第一次把 C 课上的零散知识点（指针、结构体、文件）串成一个完整项目
- 体验 EasyX 这类图形库的事件循环 + 绘制模型
- 踩了一堆指针和内存管理的坑
